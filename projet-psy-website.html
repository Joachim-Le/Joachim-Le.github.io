Psychoanalyst Website

HTML/CSS/JavaScript - 2025
Internship - 2 months
chientzu-wu-psychanalyste-paris14.fr/

  • I made this website during a 2-month Internship in 2025.
  • I developed this site for a psychoanalyst who wanted to have her own website.
  • People can find all kinds of useful information there, including:
  • A "Who am I" page describing her studies and experience,
  • Various pages describing the principles of psychoanalysis, psychiatry and psychosomatic,
  • A "Books" page showing her work translating psychoanalytic works,
  • And a "Make an appointment" page to book a face-to-face or video session.

My Work during this Internship

  • Exhaustive & Responsive Site
    • Designed the website to be responsive on all screens : smartphones, tablets and computers. Adapted the size of the header, images, maps...
    • Created a system to display the site in 3 different languages, using a combination of JavaScript code and JSON files.
    • Conceptualized a "Make an appointment" page with 2 different ways to book an appointment, using forms, Google Agenda integration, as well as a Google Maps display.
  • Meticulous Technical Aspect
    • Created a complete cookie system to let the user choose which services they consent to, with navigator storage to remember for the next visits.
    • Implemented all SEO (Search Engine Optimization) requirements for the site to be referenced on Google.

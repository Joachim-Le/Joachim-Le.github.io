Relaxed Rush

3D Puzzle Game - Unity - C# - 2024
1 student - 1½ month

  • This project is my first 3D game project.

  • It was the first project during my second year at ISART Digital.

  • It's a 3D puzzle game where you place tiles on the ground to guide cubes,
    from the start to the end of each level.

My Contributions

  • Gameplay Programming

    • Recreated an existing game from scratch using Unity 3D.

    • Coded all the gameplay, such as cube behavior and their reactions to the tiles and walls.

    • Created a dynamic UI that automatically creates itself based on the tiles available in each level.

    • Conceptualized the UX to be playable both on PC and Android smartphones/tablets.
  • Level Design

    • Designed 3 new levels in addition to the ones from the original game.

    • New levels difficulty varies from easy to hard.
  • Game Feel and Juiciness

    • Created multiple shaders to animate gameplay tiles.

    • Made several particle effects to increase juiciness.

Steam Crusher

2D Shoot'em up - Godot - C# - PC - 2023
1 student - 1½ month

  • This project is my very first work as a student at ISART Digital Paris.
  • It's a steampunk themed shoot'em up where you play as an airship.
  • Shoot the enemies, use your smoke cloud and bomb to get to the boss!
  • Experience an epic final fight with 3 phases that influence the music!

My Contributions

  • Gameplay Programming
    • Coded all the game, including :
    • Player movements and abilities.
    • Attack and movement patterns for 3 different enemies.
    • Boss patterns.
    • Parallax scrolling.
  • Game Design and Level Design
    • Designed the game from scratch following the "Steampunk" theme.
    • Conceptualized a boss battle with 3 different phases.
    • Designed weapon upgrades and collectible drops.
  • Asset and UI Creation
    • Created all assets including player(airship) and boss body parts.
    • Designed UI menus and navigation, as well as HUD updating.
    • Made visual effects for the HUD and menu buttons.

Rockoban

2D Puzzle Game - Godot - C# - 2024
5 students - 2 months

This project was part of my first year as a student at ISART Digital Paris.

My Contributions

  • Juiciness and Game Feel

    - Lights / projector effects
    - Squash and stretch effects for in-game objects
    - 2D Asset creation (box and projector sprites, reward disks)
    - Particle effects.
  • Score and Leaderboard Management

    - Score calculation based on the player's total steps.
    - View each level's leaderboard as well as the overall leaderboard.
    - Used methods to write player data in a json file, and to read data from it.
  • UI Creation

    - Created several menus for Rockoban
    - Took part in all the UI's juiciness, such as button visual effects.
    - Cooperated with one of my teammates to develop english/french localization.
